I2C_Loopback Example description

    
    This example uses the I2C Master to read/write to the I2C Slave. For 
    this example you must connect P1.6 to P3.4 and P1.7 to P3.5. The Master
    will use P3.4 and P3.5. The slave will use P1.6 and P1.7. You must also
    connect the pull-up jumpers(JP21 and JP22) to the proper I/O voltage.
    Refer to JP27 to determine the I/O voltage.
    Note Other devices on the EvKit will be using the same bus.
