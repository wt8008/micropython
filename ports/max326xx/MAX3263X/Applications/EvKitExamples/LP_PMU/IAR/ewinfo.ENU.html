LP_PMU Example description

    
    This example show how to run the PMU while the CPU is in a low power state. The PMU blinks
    LED0 every second based on the RTC's prescaler compare interrupt. Every 5 seconds the CPU
    wakes up from LP2 and blinks LED1, LED2, and LED3 in sequence.
